## **2. Buttons**

### **2.1. Primary Button**

**Mô tả:** Button chính cho các action quan trọng.

**Props:**
- `label`: Text hiển thị (required)
- `onClick`: Handler function (required)
- `disabled`: Boolean (optional, default: false)
- `loading`: Boolean (optional, default: false)
- `size`: 'sm' | 'md' | 'lg' (optional, default: 'md')
- `fullWidth`: Boolean (optional, default: false)

**States:**
- Default: Background `#2563EB`, text white
- Hover: Background `#1E40AF`, scale 1.02
- Disabled: Background `#D1D5DB`, cursor not-allowed
- Loading: Show spinner, disable interaction

**Ví dụ:**
```jsx
<Button 
  label="Đăng nhập" 
  onClick={handleLogin}
  size="md"
/>
```

### **2.2. Secondary Button**

**Mô tả:** Button phụ cho các action ít quan trọng hơn.

**Props:** Tương tự Primary Button

**States:**
- Default: Background transparent, border `#2563EB`, text `#2563EB`
- Hover: Background `#2563EB`, text white

### **2.3. Danger Button**

**Mô tả:** Button cho các action nguy hiểm (freeze account, delete).

**Props:** Tương tự Primary Button

**States:**
- Default: Background `#DC2626`, text white
- Hover: Background `#B91C1C`

**Ví dụ:**
```jsx
<Button 
  label="Đóng băng tài khoản" 
  variant="danger"
  onClick={handleFreeze}
/>
```

### **2.4. Icon Button**

**Mô tả:** Button chỉ có icon, không có text.

**Props:**
- `icon`: Icon component (required)
- `onClick`: Handler function (required)
- `ariaLabel`: Accessibility label (required)
- `size`: 'sm' | 'md' | 'lg' (optional)

## **3. Input Fields**

### **3.1. Text Input**

**Mô tả:** Input field cho text, email, password.

**Props:**
- `label`: Label text (required)
- `name`: Input name (required)
- `type`: 'text' | 'email' | 'password' | 'number' (required)
- `value`: Controlled value (required)
- `onChange`: Change handler (required)
- `placeholder`: Placeholder text (optional)
- `error`: Error message (optional)
- `required`: Boolean (optional)
- `disabled`: Boolean (optional)

**States:**
- Default: Border `#D1D5DB`
- Focus: Border `#2563EB`, outline 2px solid `#2563EB`
- Error: Border `#DC2626`, error message hiển thị phía dưới
- Disabled: Background `#F3F4F6`, cursor not-allowed

**Ví dụ:**
```jsx
<TextInput
  label="Email"
  name="email"
  type="email"
  value={email}
  onChange={handleChange}
  error={errors.email}
  required
/>
```

### **3.2. Amount Input**

**Mô tả:** Input đặc biệt cho số tiền với format VND.

**Props:** Tương tự Text Input + thêm:
- `currency`: 'VND' (default)
- `min`: Minimum value (optional)
- `max`: Maximum value (optional)

**Features:**
- Tự động format số với dấu phẩy (1,000,000)
- Hiển thị currency symbol
- Validate amount > 0

### **3.3. OTP Input**

**Mô tả:** Input cho OTP với 6 ô riêng biệt.

**Props:**
- `length`: Number of digits (default: 6)
- `onComplete`: Callback khi nhập đủ (required)
- `autoFocus`: Boolean (optional)

**Behavior:**
- Tự động focus ô tiếp theo khi nhập
- Chỉ cho phép số
- Auto-submit khi đủ số ký tự

## **7. Modals & Dialogs**

### **7.1. Modal**

**Mô tả:** Modal dialog cho confirmations, forms.

**Props:**
- `isOpen`: Boolean (required)
- `onClose`: Close handler (required)
- `title`: Modal title (required)
- `children`: Content (required)
- `footer`: Footer actions (optional)
- `size`: 'sm' | 'md' | 'lg' | 'xl' (optional)

**Behavior:**
- Backdrop click để đóng (optional)
- ESC key để đóng
- Focus trap
- Scroll lock khi mở

**Ví dụ:**
```jsx
<Modal
  isOpen={isOpen}
  onClose={handleClose}
  title="Xác nhận đóng băng tài khoản"
  footer={
    <>
      <Button label="Hủy" variant="secondary" onClick={handleClose} />
      <Button label="Xác nhận" variant="danger" onClick={handleConfirm} />
    </>
  }
>
  <p>Bạn có chắc chắn muốn đóng băng tài khoản?</p>
</Modal>
```

### **7.2. Confirmation Dialog**

**Mô tả:** Dialog đơn giản cho confirmations.

**Props:**
- `isOpen`: Boolean (required)

- `confirmLabel`: Confirm button text (optional, default: "Xác nhận")
- `cancelLabel`: Cancel button text (optional, default: "Hủy")
