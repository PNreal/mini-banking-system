## **11. Visual Hierarchy**

### **11.1. Information Priority**

1. **Primary**: Balance, Account Status

### **11.2. Visual Weight**

- **Heavy**: Primary actions, Important info
- **Medium**: Secondary actions, Regular content
- **Light**: Tertiary actions, Supporting content

## **12. Spacing Guidelines**

### **12.1. Component Spacing**

- Between cards: 16px (mobile), 24px (desktop)
- Between sections: 32px (mobile), 48px (desktop)
- Card padding: 16px (mobile), 24px (desktop)

### **12.2. Content Spacing**

- Between form fields: 16px
- Between label and input: 8px
- Between button groups: 12px

- Medium-sized components

- Multi-column grid
- Full sidebar
- Larger components
- More whitespace
