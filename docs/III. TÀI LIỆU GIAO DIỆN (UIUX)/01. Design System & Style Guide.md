## **1. Tổng quan**

Tài liệu này định nghĩa hệ thống thiết kế (Design System) cho Mini Banking System, bao gồm màu sắc, typography, spacing, shadows, và các quy tắc thiết kế cơ bản.

## **4. Spacing System**

Hệ thống spacing dựa trên bội số của 4px:

| Tên | Size | Sử dụng |
|-----|------|---------|
| **xs** | 4px | Tight spacing trong components |
| **sm** | 8px | Spacing nhỏ giữa elements |
| **md** | 16px | Spacing chuẩn giữa elements |
| **lg** | 24px | Spacing lớn giữa sections |
| **xl** | 32px | Spacing rất lớn giữa major sections |
| **2xl** | 48px | Spacing cực lớn |

### **4.1. Component Spacing**

- **Card Padding**: 24px (lg)
- **Button Padding**: 12px vertical, 24px horizontal
- **Input Padding**: 12px vertical, 16px horizontal
- **Section Margin**: 32px (xl)

## **10. Animation & Transitions**

### **10.1. Duration**

| Tên | Duration | Sử dụng |
|-----|----------|---------|
| **Fast** | 150ms | Hover states, quick feedback |
| **Normal** | 200ms | Standard transitions |
| **Slow** | 300ms | Complex animations |

### **10.2. Easing**

- **Ease In**: `cubic-bezier(0.4, 0, 1, 1)`
- **Ease Out**: `cubic-bezier(0, 0, 0.2, 1)`
- **Ease In Out**: `cubic-bezier(0.4, 0, 0.2, 1)`

### **10.3. Common Animations**

- **Button Hover**: Scale 1.02, transition 150ms
- **Card Hover**: Shadow elevation, transition 200ms
- **Modal Enter**: Fade in + slide up, 300ms
- **Toast Notification**: Slide in from right, 300ms

## **11. Accessibility Standards**

### **11.1. Color Contrast**

- **Text trên Background**: Tối thiểu 4.5:1 (WCAG AA)
- **Large Text**: Tối thiểu 3:1 (WCAG AA)
- **Interactive Elements**: Tối thiểu 3:1

### **11.2. Focus States**

- **Focus Outline**: 2px solid Primary Blue
- **Focus Offset**: 2px từ element
- **Keyboard Navigation**: Tất cả interactive elements phải có focus state

### **11.3. Touch Targets**

- **Minimum Size**: 44x44px (iOS/Android standard)
- **Spacing**: Tối thiểu 8px giữa các touch targets

## **13. Examples**

### **13.1. Button Styles**

```css
/* Primary Button */
.button-primary {
  background-color: #2563EB;
  color: white;
  padding: 12px 24px;
  border-radius: 8px;
  font-size: 14px;
  font-weight: 500;
  transition: all 150ms;
}

.button-primary:hover {
  background-color: #1E40AF;
  transform: scale(1.02);
}

/* Danger Button */
.button-danger {
  background-color: #DC2626;
  color: white;
  /* ... */
}
```

### **13.2. Card Styles**

```css
.card {
  background: white;
  border-radius: 12px;
  padding: 24px;
  box-shadow: 0 4px 6px -1px rgba(0, 0, 0, 0.1);
  transition: box-shadow 200ms;
}

.card:hover {
  box-shadow: 0 10px 15px -3px rgba(0, 0, 0, 0.1);
}
```
